Játékmotorok

Kezdőknek ajánlott

  • GameMaker: nagy múltú, kiforrott 2D engine. Ingyenes, de kereskedelmi forgalomba kerülő játékok fejlesztéséhez licencet kell venni egyszeri 40 000 Ft körüli összegért.
  • Construct: freemium 2D engine (korlátozott 3D funkciókkal), ami böngészőben fut, de offline is működik. Esemény-orientált vizuális módszerrel, vagy JavaScript ill. TypeScript nyelven programozható. Az ingyenes verzió kipróbálásra éppen elég, de meglehetősen korlátozott. A korlátlan változat havi vagy éves előfizetési díjért vehető igénybe (az éves díja kb. 45 000 Ft magánszemélyeknek).
  • GDevelop: A Construct-ra nagyon sok szempontból hasonlító ingyenes engine.
  • microStudio: Nyílt forrású 2D engine, böngészős és letölthető verziója is van. Saját (Lua-szerű) microScript nyelvén, valamint Lua, Python, Javascript, ill. Typescript nyelven is programozható.
  • GB Studio: nyílt forrású fejlesztőkörnyezet GameBoy / GameBoy Color játékok készítéséhez. A játékokból böngészőben futó verziót is lehet exportálni, de akár GameBoy hardveren vagy emulátoron futó verziót is. Blokk-alapú vizuális módszerrel programozható.
  • Ct.js: nyílt forrású 2D engine integrált fejlesztői környezettel, JavaScript nyelven vagy a Catnip nevű blokk-alapú vizuális módszerrel programozható.
  • Twine: nyílt forrású engine interaktív történetek, szöveges kalandjátékok és társaik létrehozásához. Böngészőben is használható, kódolási ismereteket nem igényel, de van benne CSS / JavaScript támogatás is.
  • Bitsy: nyílt forrású 2D engine elsősorban retro stílusú játékok létrehozásához, kódolást nem igényel.
  • Whimsy: ingyenes 2D engine, amit a Bitsy inspirált. Letölthető és böngészőben futó verziója is van, vizuális módszerrel programozható.
  • RPG in a Box: egyszeri 8000 Ft körüli összegért megvásárolható 2D/3D engine elsősorban RPG-k készítéséhez. Vizuális módszerrel, vagy a saját Bauxite nevű script nyelvén keresztül programozható.
  • RPG Maker MZ / MV / VX / XP / etc: nagy múltú engine család 2D RPG-k készítéséhez. A legtöbb változat Steam akciók keretében néhány ezer Forintért beszerezhető, de mindegyikből van 30 napos ingyenes próbaverzió is.
  • RPG Paper Maker: az RPG Maker-re némileg hajazó nyílt forrású 2.5D engine, ami hobbi célra ingyenesen használható. Kereskedelmi forgalomba kerülő játékok készítéséhez licenc vásárlása szükséges, ami kb. 27 000 Ft.
  • PuzzleScript: nyílt forrású, böngészőben futó 2D játékmotor és script nyelv elsősorban egyszerű, retro stílusú puzzle és akciójátékok fejlesztéséhez.

A nagy nevek

  • Godot: népszerű nyílt forrású 2D/3D engine. A GDScript nevű saját (Python-szerű) nyelvén lehet programozni, de van C# verziója is (ami jelenleg kevésbé elterjedt).
  • Unity: alighanem a legnépszerűbb freemium 2D/3D engine, rengeteg oktatóanyagot lehet találni hozzá a neten. C# nyelven, vagy a gráf alapú Visual Scripting rendszer segítségével programozható.
  • Unreal Engine: ingyenes, elsősorban 3D engine. C++ nyelven vagy a gráf alapú Blueprint rendszer segítségével programozható.

Még néhány ajánlás

  • LÖVE: nyílt forrású 2D engine, Lua nyelven programozható, de integrált fejlesztőkörnyezet "gyárilag" nem tartalmaz.
  • PICO-8: egyszeri 6000 Ft körüli összegért megvehető "fantasy console" és fejlesztőeszköz. Lua nyelven lehet elsősorban retro stílusú játékokat készíteni vele.
  • TIC-80: nyílt forrású "fantasy console", ami sok tekintetben hasonlít a PICO-8-re.
  • quadplay✜: nyílt forrású "fantasy console", amit egy PyxlScript nevű Python-szerű nyelven lehet programozni.

2D / 3D grafika és animáció

2D rajz, pixel art és animáció

  • Pixel Studio: ingyenes pixel art app ami Windows, Mac, Android és iPadOS platformokra is elérhető, animálni is lehet vele.
  • Aseprite: népszerű nyílt forrású pixel art app, ami minden népszerű desktop platformon elérhető, egyszeri 7000 Ft körüli díjért vásárolható meg, de ha valaki veszi a fáradságot, forrásból lefordíthatja és akkor ingyen van.
  • Resprite: az Aseprite-ra sokban hasonlító ingyenes app, ami desktop és Android / iPadOS platformokra is elérhető.
  • Pixelorama: böngészőben és desktop platformokon is elérhető ingyenes pixel art app.
  • Photopea: böngészőben futó ingyenes (de reklámokat megjelenítő) képszerkesztő program erős Phtoshop utánérzéssel.
  • Piskel: böngészőben futó nyílt forrású pixel art app.
  • Krita: nyílt forrású digitális rajzolóprogram, Windows, Mac és Linux platformokra is elérhető.
  • Pixelable: ingyenes pixel art app iPadOS-re. Reklámmentességet egyszeri 1500 Ft körüli összegért lehet venni benne.
  • Pixaki: pixel art app iPadOS-re, a demo verzió ingyenesen kipróbálható, a teljesért egyszeri 7000 Ft körüli összeget kell fizetni.
  • FlipaClip: 2D képkockánkénti animációhoz kitalált intuitív freemium program Android és iOS eszközökre.
  • PixiEditor: nyílt forrású 2D grafikai program Windows, Mac és Linux platformokra számos funkcióval (képszerkesztés, pixelgrafika, vektorgrafika, animáció, procedurális generálás).

Vektorgrafika

  • Inkscape: nyílt forrású vektorgrafikus program Windows, Mac és Linux platformokra.
  • Graphite: nyílt forrású, böngészőben futó vektorgrafikus program. Aktív fejlesztés alatt áll, de az alpha verzió már kipróbálható.
  • Rive: freemium böngészőben futó vektorgrafikus program. Animálni is lehet vele, számos játékmotort is támogat.

3D modellezés és voxel szerkesztők

  • MagicaVoxel: nyílt forrású voxel art program Windows és Mac platformokra.
  • Blockbench: nyílt forrású voxel art / low poly 3D modellező program Windows, Mac és Linux platformokra.
  • Goxel: nyílt forrású voxel art program, Windows, Mac, Linux, Android és iPadOS platformokra.
  • Blender: a legnépszerűbb nyílt forrású 3D modellező program, elérhető Windows, Mac és Linux platformokra is. Mivel rengeteg funkció van benne, kissé nehéz lehet elsőre eligazodni benne, de sok tutorial elérhető hozzá.
  • Tinkercad: böngészőben futó ingyenes alacsony küszöbű 3D modellező program.
  • Kenney Shape: Windows és Linux platformokra elérhető program, mellyel 2D pixel artból lehet 3D modelleket extrudálni. Egyszeri 1500 Ft körüli összegért lehet megvásárolni.
  • Crocotile 3D: elsősorban környezetek / pályák modellezésére kitalált 3D szerkesztő program Windows, Mac és Linux platformokra. A demó ingyenes, a teljes verzió egyszeri 11 000 Ft körüli összegért vásárolható meg.
  • InstaMAT: freemium eszköz 3D modellek textúráinak készítéséhez.

Audio programok

Hang- és zeneszerkesztő szoftverek

  • Audacity / Tenacity: az Audacity egy kicsit cseles háttérsztorival rendelkező nyílt forrású destruktív hangszerkesztő program; a Tenacity pedig egy kevésbé cseles forkja, ami funkciókban némileg eltér az eredetitől. Mindkettő támogatja a VST(3) effekteket.
  • REAPER: elsősorban zenére kihegyezett, de hangeffektek készítésére / manipulálására is kiváló nem-destruktív szerkesztőprogram (DAW). Ingyenesen korlátozások nélkül 60 napig kipróbálható, de 60 nap után sem korlátozza a funkciókat. Ugyanakkor érdemes leszurkolni a 22 000 Ft körüli egyszeri összeget érte, kevés jobb ár-érték arányú szoftver van a világon. Támogatja a VST hangszereket és effekteket is, néhány beépített effekteket is tartalmaz.
  • FL Studio: fizetős, de rég óta népszerű zeneszerkesztő. A belépő verzió 35 000 Ft körüli egyszeri díjért vásárolható meg.
  • LMMS: nyílt forrású, Windows, Linux és Mac rendszerekre is elérhető szoftver, ami látszólag elég sokat inspirálódott az FL Studio-ból.
  • Tracktion Waveform: korlátozott ingyenes verzióval is rendelkező, Windows, Linux és Mac rendszereken is futó zeneszerkesztő szoftver.
  • Ableton Live: 30 napos próbaverzióval rendelkező népszerű zeneszerkesztő szoftver. A belépő verzió 32 000 Ft körüli összegért vásárolható meg.
  • Bandlab Studio: freemium, böngészőben és Android/iOS eszközökön futó zeneszerkesztő.
  • Cakewalk by Bandlab: freemium, Windows és Mac platformokra elérhető zeneszerkesztő.
  • BeepBox: böngészőben futó (de offline is működő) nyílt forrású szekvenszer beépített hangszerekkel.
  • DrawBeats: ingyenes, böngészőben futó szekvenszer, kezdőknek is ideális.
  • OpenMPT: nyílt forrású tracker VST effekt és hangszer támogatással.
  • Bassoon Tracker: nyílt forrású, böngészőben futó tracker.
  • Bosca Ceoil: The Blue Album: nyílt forrású, több platformon is elérhető kezdőbarát szekvenszer beépített hangszerekkel.

VST hangszerek és effektek

  • IK Multimedia és Native Instruments: jó pár ingyenes effektjük és hangszerük is van, de viszonylag gyakran akcióznak is, amikor nagyon jó áron lehet hozzájutni a fizetős cuccaikhoz.
  • Plugins4Free: ingyenes effekteket és hangszereket gyűjtő oldal.

Hangeffekt generátorok

  • sfxr, bfxr, jfxr, jsfxr: egy családból származó nyílt forrású parametrikus hangeffekt-szintetizátorok.
  • Chiptone: A fentiekhez hasonló, de még több funkcióval rendelkező hangeffekt-szintetizátor.
  • LabChirp: ingyenes, Windows, Mac és Linux platformokra is letölthető hangeffekt-szintetizátor.

Asset lelőhelyek (2D / 3D grafika, hangeffektek, zene)

  • Itch.io: rengeteg ingyenes és fizetős 2D/3D grafikai asset, hangeffekt és zene is beszerezhető az Itch piacterén.
  • OpenGameArt: ingyenes, részben szabad licencelésű asset gyűtőoldal: iss is van minden.
  • Kenney, avagy Asset Jézus: egy szuper jó fej holland arc oldala, ahonnan kilátástalan mennyiségű, minőségi CC0 licencelt assetet lehet letölteni. Vannak generátor / szerkesztő programjai és példaprojektjei is.
  • Quaternius: CC0 licenc alatt elérhető alacsony poligonszámú minőségű 3D modellek.
  • PolyPizza: ingyenes, részben szabad licencelésű 3D modelleket gyűjtő oldal.
  • Share Textures: CC0 licencelt textúrákat gyűjtő oldal.
  • Poly Haven és ambientCG: CC0 licencelt modelleket és textúrákat gyűjtő oldalak.
  • 99Sounds: ingyenes hangeffekt-csomagok.
  • Lospec: palettákat és pixel art tutorialokat gyűjtő oldal.
  • Incompetech: Kevin MacLeod CC-BY licencelt zenéinek lelőhelye.
  • Freesound: ingyenes (és sok esetben szabad licencelésű) hangeffekteket gyűjtő oldal. Letöltéshez regisztráció szükséges.
  • FontSpace: ingyenes betűtípusokat gyűjtő oldal, ahol lehet szűrni kereskedelmi célra használható fontokra is.
  • Google Fonts: ingyenes betűtípusokat gyűjtő oldal
  • Google Webfonts Helper: alternatív Google Fonts frontend, ami lehetővé teszi a betűtípusok letöltését több fájlformátumban is (woff, woff2, ttf, otf, eot, svg).
  • Pixabay: ingyenes és kedvező licencelésű assetek gyűjtőhelye. 2D raszter- és vektorgrafika, 3D modellek, zene és hangeffektek is vannak fent.

Kapcsolódó témakörök

Terjesztés

  • Itch.io: az Itch részben közösségi oldal játékfejlesztőknek, részben egy piactér, ahol játékokat és asseteket lehet árulni és vásárolni (de van rengeteg ingyenes cucc is fent). Fejlesztők ingyenesen feltölthetik a játékaikat (böngészős és letölthető verziókat is), van game jam gyűjtőoldal, meg még ezer másik hasznos dolog.
  • Github Pages, Gitlab Pages, Vercel: ingyenes opciók böngészőben futó játékok közzétételére (bizonyos korlátok mellett).

Projekt tervezés, kollaboráció

Online eszközök ötleteléshez, tervezéshez, dokumentáláshoz, a játék állapotának követéséhez, hibakezeléshez és hasonlókhoz.

  • Trello: kanban-típusú (sávokat és kártyákat használó) freemium projektmenedzsment eszköz.
  • Miro: "végtelen tábla"-típusú freemium kollaborációs eszköz.
  • Notion: freemium eszköz kollaboratív tervezéshez.
  • Obsidian, Logseq: népszerű nyílt forrású programok tervezéshez és dokumentáláshoz.

Gamedev továbbképzés

  • Red Blob Games: ingyenes interaktív magyarázó anyagok videójátékokban használatos technikákról.
  • Game UI Database: videójátékok felhasználói felületét dokumentáló és elemző gyűjtőoldal.
  • Coffee with Butterscotch: a világ alighanem legjobb gamedev témát is sűrűn érintő, egyébként szupervicces podcastje. A Butterscotch Shenanigans csapatától rengeteget lehet tanulni. A történetüket is ajánlott megismerni.
  • Pixel Art Tutorials: Pedro Medeiros (Saint11) elképesztően menő animált GIF-ekbe öntött pixel art tutorialjai.
  • Omnicalculator: matematikai képletek leírása és interaktív kalkulátorok.
  • PoseMy.Art: emberszabású karakterek rajzolásához / animálásához hasznos pózok gyűjteménye (referenciának)
  • Game Feel Demo (Deepnight Games): interaktív demo, ami egy rakás játék "juice" technikát mutat be.

Programozás

Ha általánosságban szeretnéd programozási ismereteidet bővíteni, nézz körül ezeken az oldalakon.

  • freeCodeCamp: ingyenes tananyagok több témában.
  • Roadmap.sh: ingyenes tananyagok több témában.
  • Future Coder: ingyenes, modern, interaktív Python oktatóanyag.
  • W3Schools: ingyenes oktatóanyagok főleg (de nem kizárólag) webfejlesztéshez kapcsolódó témákban.

Szolgáltatások

  • Free for Developers: linkgyűjtemény fejlesztőknek, akik szeretik az ingyenes szolgáltatásokat / termékeket :)
